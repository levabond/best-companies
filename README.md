### Best Companies

Очень простое приложение на простых технологиях со списком компаний

Для работы потребуется генератор:
gem install generamba
- https://github.com/strongself/Generamba

**Реализация:**
- Архитектура VIPER
- Верстка SnapKit
- Запросы через URLSession и clousers
